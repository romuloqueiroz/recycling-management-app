# Recycling Management App

## Vision

Das Projekt soll das lokale Recycling-Management durch eine moderne Webanwendung verbessern.

## Hauptfunktionen

- Recyclingstationen in der Nähe finden
- Akzeptierte Materialien anzeigen
- Benutzerregistrierung
- Nachverfolgung persönlicher Recyclingbeiträge

## Mitwirken

Interessierte Entwickler können durch Issues und Pull Requests zum Projekt beitragen.
